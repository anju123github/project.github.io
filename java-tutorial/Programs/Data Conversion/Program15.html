15. How to convert int to long

Code


                public class IntToLongConversion {
                    public static void main(String[] args) {
                        // Convert int to long
                        int intValue = 42;
                        long longValue = intValue;
                
                        // Print the results
                        System.out.println("Original int value: " + intValue);
                        System.out.println("Converted to long value: " + longValue);
                    }
                }

Output

Original int value: 42
Converted to long value: 42
