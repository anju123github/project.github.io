16. How to convert long to int

Code


                public class LongToIntConversion {
                    public static void main(String[] args) {
                        // Convert long to int with explicit casting
                        long longValue = 9876543210L; // Make sure to use the 'L' suffix for long literals
                        int intValue = (int) longValue;
                
                        // Print the results
                        System.out.println("Original long value: " + longValue);
                        System.out.println("Converted to int value: " + intValue);
                    }
                }

Output

Original long value: 9876543210
Converted to int value: 1215752194
