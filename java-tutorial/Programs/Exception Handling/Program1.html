Exception Handling

1. Write a Java program that throws an exception and catch it using a try-catch block.

Code


                    public class ExceptionExample {
                    public static void main(String[] args) {
                        try {
                            // Code that may throw an exception
                            int result = divide(10, 0);
                                                        
                            // This line won't be executed if an exception occurs
                            System.out.println("Result: " + result);
                        } catch (ArithmeticException e) {
                            // Catch the specific exception (ArithmeticException in this case)
                            System.out.println("Exception caught: " + e.getMessage());
                        } finally {
                            // This block will be executed regardless of whether an exception occurs
                            System.out.println("Finally block executed");
                        }
                                                        
                        // Rest of the program continues after handling the exception
                        System.out.println("Program continues...");
                    }
                                                        
                    // Function that may throw an exception
                    private static int divide(int numerator, int denominator) {
                        // Attempt to divide by zero, which will result in an ArithmeticException
                        return numerator / denominator;
                    }
                }

Output
