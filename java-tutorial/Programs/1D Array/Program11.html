11. Program to print the number of elements present in an array

Code


                public class NumberOfElementsExample {
                    public static void main(String[] args) {
                        int[] array = {12, 45, 78, 23, 56, 89, 34, 67};
                                                        
                        System.out.println("Original Array:");
                        printArray(array);
                                                        
                        int numberOfElements = array.length;
                        System.out.println("\nNumber of Elements in the Array: " + numberOfElements);
                    }
                                                        
                    private static void printArray(int[] array) {
                        for (int value : array) {
                            System.out.print(value + " ");
                        }
                    }
                }

Output

Original Array: 

12 45 78 23 56 89 34 67  

Number of Elements in the Array: 8
