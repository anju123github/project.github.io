11. Write a Java method that accepts three integers and checks whether they are consecutive or not. Returns true or false.

Code


                public class ConsecutiveChecker {

                    // Method to check if three integers are consecutive
                    public static boolean areConsecutive(int num1, int num2, int num3) {
                        // Check if the difference between consecutive numbers is 1
                        return (num2 == num1 + 1 && num3 == num2 + 1);
                    }
                                                
                    public static void main(String[] args) {
                        // Example usage:
                        int num1 = 4;
                        int num2 = 5;
                        int num3 = 6;
                                                
                        System.out.println("Are " + num1 + ", " + num2 + ", and " + num3 + " consecutive? " + areConsecutive(num1, num2, num3));
                    }
                }

Output

    Are 4, 5, and 6 consecutive? true
