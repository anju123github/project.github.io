3. Write a Java method to display the middle character of a string. Note: a) If the length of the string is odd there will be two middle characters. b) If the length of the string is even there will be one middle character.

Code


                public class MiddleCharacterDisplay {

                    // Method to display the middle character(s) of a string
                    public static String displayMiddleCharacter(String input) {
                        int length = input.length();
                
                        if (length == 0) {
                            return "String is empty.";
                        }
                
                        int middleIndex = length / 2;
                
                        if (length % 2 == 0) {
                            // String has an even length, so there is one middle character
                            return "Middle character: " + input.charAt(middleIndex - 1) + input.charAt(middleIndex);
                        } else {
                            // String has an odd length, so there are two middle characters
                            return "Middle character: " + input.charAt(middleIndex);
                        }
                    }
                
                    public static void main(String[] args) {
                        // Example usage:
                        String str1 = "abcde";
                        String str2 = "abcdef";
                        String str3 = "";
                
                        System.out.println(displayMiddleCharacter(str1));
                        System.out.println(displayMiddleCharacter(str2));
                        System.out.println(displayMiddleCharacter(str3));
                    }
                }

Output

Middle character: c 

Middle character: de 

String is empty.
