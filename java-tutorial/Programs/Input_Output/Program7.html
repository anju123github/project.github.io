7. Enter the short integer from user using Scanner class

Code


                
import java.util.Scanner;

public class ReadShortFromUser {
    public static void main(String[] args) {
        // Create a Scanner object to read input from the console
        Scanner scanner = new Scanner(System.in);
                                
        try {
            // Prompt the user to enter a short integer
            System.out.print("Enter a short integer: ");
                                
            // Read the short integer from the user
            short userInput = scanner.nextShort();
                                
            // Print the entered short integer
            System.out.println("You entered: " + userInput);
        } catch (java.util.InputMismatchException e) {
            System.err.println("Error: Please enter a valid short integer.");
        } finally {
            // Close the Scanner
            scanner.close();
        }
    }
}

Output

    Enter a short integer: 32767 

    You entered: 32767
