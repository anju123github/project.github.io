Multithreading

10. Create the program to create multiple threads, set its name and its priority.

Code


                
class MyThread extends Thread {
    public MyThread(String name, int priority) {
        super(name);
        setPriority(priority);
    }

    public void run() {
        System.out.println("Thread " + getName() + " with priority " + getPriority() + " is starting");
        
        // Simulate some work
        try {
            Thread.sleep(2000);
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

        System.out.println("Thread " + getName() + " is exiting");
    }
}

public class ThreadExample {
    public static void main(String[] args) {
        // Create multiple threads
        MyThread thread1 = new MyThread("Thread-1", Thread.MAX_PRIORITY);
        MyThread thread2 = new MyThread("Thread-2", Thread.MIN_PRIORITY);

        // Start the threads
        thread1.start();
        thread2.start();

        // Wait for threads to finish
        try {
            thread1.join();
            thread2.join();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }
    }
}

Output

Thread Thread-1 with priority 10 is starting
Thread Thread-2 with priority 1 is starting
Thread Thread-2 is exiting
Thread Thread-1 is exiting
