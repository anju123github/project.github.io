Packages

8.Create a Java program to create a package with public class and protected members to be accessed in another class.

Code


                // File: MainApp.java

                package MyPackage;
                
                public class MainApp {
                    public static void main(String[] args) {
                        MyPackageClass myPackageObj = new MyPackageClass();
                
                        // Accessing protected members from another class
                        System.out.println("Protected Int: " + myPackageObj.protectedInt);
                        System.out.println("Protected String: " + myPackageObj.protectedString);
                    }
                
                    public static class MyPackageClass {
                        protected int protectedInt = 10;
                        protected String protectedString = "Hello, protected world!";
                    }
                }

Output

    Protected Int: 10
    Protected String: Hello, protected world!
