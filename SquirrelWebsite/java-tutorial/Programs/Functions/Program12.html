12. Write a Java method that accepts three integers and returns true if one is the middle point between the other two integers, otherwise false

Code


                public class MiddlePointChecker {

                    // Method to check if one integer is the middle point between the other two
                    public static boolean isMiddlePoint(int num1, int num2, int num3) {
                        // Sort the numbers
                        int[] sortedNumbers = {num1, num2, num3};
                        java.util.Arrays.sort(sortedNumbers);
                                                
                        // Check if the middle number is the middle point
                        return (sortedNumbers[1] == num2);
                    }
                                                
                    public static void main(String[] args) {
                        // Example usage:
                        int num1 = 3;
                        int num2 = 6;
                        int num3 = 9;
                                                
                        System.out.println("Is " + num2 + " the middle point between " + num1 + " and " + num3 + "? " + isMiddlePoint(num1, num2, num3));
                    }
                }

Output

    Is 6 the middle point between 3 and 9? true
