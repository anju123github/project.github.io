8. Enter the long integer from user using Scanner class

Code


import java.util.Scanner;

public class ReadLongFromUser {
    public static void main(String[] args) {
        // Create a Scanner object to read input from the console
        Scanner scanner = new Scanner(System.in);
                                
        try {
            // Prompt the user to enter a long integer
            System.out.print("Enter a long integer: ");
                                
            // Read the long integer from the user
            long userInput = scanner.nextLong();
                                
            // Print the entered long integer
            System.out.println("You entered: " + userInput);
        } catch (java.util.InputMismatchException e) {
            System.err.println("Error: Please enter a valid long integer.");
        } finally {
            // Close the Scanner
            scanner.close();
        }
    }
}

Output

Enter a long integer: 123456789012345 

You entered: 123456789012345
