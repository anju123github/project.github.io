Inheritance

6. Write a Java program to create a class called Animal with a method named move(). Create a subclass called Cheetah that overrides the move() method to run.

Code


                
// Base class Animal
class Animal {
    // Method to perform movement
    public void move() {
        System.out.println("Animal is moving");
    }
}
                                        
// Subclass Cheetah extending Animal
class Cheetah extends Animal {
    // Override the move method for Cheetah
    
    public void move() {
        System.out.println("Cheetah is running");
    }
}
                                        
// Main class
public class AnimalExample {
    public static void main(String[] args) {
        // Create an instance of Animal
        Animal genericAnimal = new Animal();

        // Create an instance of Cheetah
        Cheetah myCheetah = new Cheetah();

        // Demonstrate move for both instances
        genericAnimal.move();  // Output: Animal is moving
        myCheetah.move();      // Output: Cheetah is running
    }
}

Output

Animal is moving 

Cheetah is running
