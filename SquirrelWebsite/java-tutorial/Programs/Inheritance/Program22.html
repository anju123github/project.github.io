Inheritance

22. Write a Java program to create an abstract class Person with abstract methods eat() and exercise(). Create subclasses Athlete and LazyPerson that extend the Person class and implement the respective methods to describe how each person eats and exercises

Code


                
// Abstract class Person
abstract class Person {
    // Abstract methods
    public abstract void eat();
    public abstract void exercise();
}
                                        
// Subclass Athlete extending Person
class Athlete extends Person {
    // Implementing the eat method for Athlete
    
    public void eat() {
        System.out.println("Athlete eats a balanced diet for energy");
    }
                                        
    // Implementing the exercise method for Athlete
    
    public void exercise() {
        System.out.println("Athlete engages in rigorous exercise routines");
    }
}
                                        
// Subclass LazyPerson extending Person
class LazyPerson extends Person {
    // Implementing the eat method for LazyPerson
    
    public void eat() {
        System.out.println("Lazy person eats snacks and fast food");
    }
                                        
    // Implementing the exercise method for LazyPerson
    
    public void exercise() {
        System.out.println("Lazy person avoids exercise and prefers a sedentary lifestyle");
    }
}
                                        
// Main class
public class AbstractPersonExample {
    public static void main(String[] args) {
        // Create instances of Athlete and LazyPerson
        Athlete athlete = new Athlete();
        LazyPerson lazyPerson = new LazyPerson();
                                        
        // Describe eating and exercise habits for Athlete and LazyPerson
        System.out.println("Athlete:");
        athlete.eat();
        athlete.exercise();
                                        
        System.out.println(); // Empty line for clarity
                                        
        System.out.println("Lazy Person:");
        lazyPerson.eat();
        lazyPerson.exercise();
    }
}

Output

Athlete: 

Athlete eats a balanced diet for energy 

Athlete engages in rigorous exercise routines 

 

Lazy Person: 

Lazy person eats snacks and fast food 

Lazy person avoids exercise and prefers a sedentary lifestyle
