Inheritance

7. Write a Java program to create a class known as Person with methods called getFirstName() and getLastName(). Create a subclass called Employee that adds a new method named getEmployeeId() and overrides the getLastName() method to include the employee's job title

Code


                
              // Base class Person
              class Person {
                  // Attributes
                  private String firstName;
                  private String lastName;
                                                      
                  // Constructor
                  public Person(String firstName, String lastName) {
                      this.firstName = firstName;
                      this.lastName = lastName;
                  }
                                                      
                  // Method to get the first name
                  public String getFirstName() {
                      return firstName;
                  }
                                                      
                  // Method to get the last name
                  public String getLastName() {
                      return lastName;
                  }
              }
                                                      
              // Subclass Employee extending Person
              class Employee extends Person {
                  // Attributes specific to Employee
                  private int employeeId;
                  private String jobTitle;
                                                      
                  // Constructor for Employee
                  public Employee(String firstName, String lastName, int employeeId, String jobTitle) {
                      super(firstName, lastName);
                      this.employeeId = employeeId;
                      this.jobTitle = jobTitle;
                  }
                                                      
                  // New method specific to Employee
                  public int getEmployeeId() {
                      return employeeId;
                  }
                                                      
                  // Override the getLastName method to include the employee's job title
                  
                  public String getLastName() {
                      return super.getLastName() + ", " + jobTitle;
                  }
              }
                                                      
              // Main class
              public class PersonExample {
                  public static void main(String[] args) {
                      // Create an instance of Person
                      Person genericPerson = new Person("John", "Doe");
              
                      // Create an instance of Employee
                      Employee myEmployee = new Employee("Alice", "Smith", 1234, "Software Engineer");
              
                      // Demonstrate getFirstName, getLastName, and getEmployeeId for both instances
                      System.out.println("Person: " + genericPerson.getFirstName() + " " + genericPerson.getLastName());
                      System.out.println("Employee: " + myEmployee.getFirstName() + " " + myEmployee.getLastName());
                      System.out.println("Employee ID: " + myEmployee.getEmployeeId());
                  }
              }

Output

Person: John Doe 

Employee: Alice Smith, Software Engineer 

Employee ID: 1234
