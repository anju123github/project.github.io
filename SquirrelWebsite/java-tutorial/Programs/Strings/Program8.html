8. Program to Add Characters to a String

Code


                public class AddCharactersToString {
                    public static void main(String[] args) {
                        // Original string
                        String originalString = "Hello, ";
                
                        // Characters to add
                        String charactersToAdd = "World!";
                
                        // Using the + operator to add characters
                       String modifiedString = originalString + charactersToAdd;
                
                        // Display the result
                        System.out.println("Modified String: " + modifiedString);
                    }
                }

Output

    Modified String: Hello, World!
