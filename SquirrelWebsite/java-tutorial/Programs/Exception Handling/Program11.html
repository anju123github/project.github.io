Exception Handling

11. Write a Java program to show an example of throws Keyword.

Code


                import java.io.IOException;

                public class ThrowsExample {
                public static void main(String[] args) {
                        try {
                            exampleMethod();
                        } catch (IOException e) {
                            System.out.println("Caught IOException: " + e.getMessage());
                        }
                    }
                                                        
                    // Using 'throws' in the method signature to declare that it may throw an IOException
                    private static void exampleMethod() throws IOException {
                        System.out.println("Inside exampleMethod");
                        // Simulating an IOException
                        throw new IOException("This is a simulated IOException");
                    }
                }

Output

Inside exampleMethod 

Caught IOException: This is a simulated IOException
