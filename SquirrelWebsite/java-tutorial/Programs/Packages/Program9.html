Packages

9.Create a Java program to create a nested package i.e. c4.sample with public class and protected members to be accessed in another class.

Code


                // File: MainApp.java

                package c4;
                
                public class MainApp {
                    public static void main(String[] args) {
                        c4.sample.SampleClass sampleObj = new c4.sample.SampleClass();
                
                        // Accessing protected members from another class
                        System.out.println("Protected Int: " + sampleObj.protectedInt);
                        System.out.println("Protected String: " + sampleObj.protectedString);
                    }
                
                    public static class SampleClass {
                        protected int protectedInt = 42;
                        protected String protectedString = "Hello, protected world!";
                    }
                }

Output

    Protected Int: 42
    Protected String: Hello, protected world!
