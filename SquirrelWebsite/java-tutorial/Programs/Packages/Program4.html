4.Create a Java program to access the java.util.Date class through qualified way and print the current date of the system but inherit the Date class.

Code


                // File: CustomDate.java

                import java.util.Date;
                
                public class CustomDate extends Date {
                    public static void main(String[] args) {
                        // Accessing the current date through the qualified way
                        CustomDate customDate = new CustomDate();
                        
                        // Printing the current date
                        System.out.println("Current Date: " + customDate);
                    }
                }

Output

    Current Date: Mon Nov 16 12:34:56 UTC 2023
